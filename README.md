# Kagi Mac OS 9 Platinum

<img width="1920" height="1080" alt="image" src="https://github.com/user-attachments/assets/c65e70ee-9819-466f-8908-b59f8f16d077" />

A retro and nostalgia-inducing custom theme for Kagi that mimicks the look and feel of Mac OS 9 Platinum released in 1999.

This is still being worked on, so there are bugs!

## Installation
1. Go to Appearance
2. Change theme to "Default" not "Light" or "Dark"
3. Enable Custom CSS & Paste the entirety of custom.css
4. Save changes and profit?

# Features

- The footer has been moved to create the iconic Apple menu bar on the top
- Follows most of the Mac OS 9 styling for a modern age
- Most elements have been skinned
- Is responsive to desktop, tablet and mobile
- Automatically changes to dark or light mode based on system
